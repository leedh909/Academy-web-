***목록 관련 태그

//항목 추가는 <li></li> 

*순서 있는 목록 태그
//<ol>항목들</ol>
//항목 추가는 <li></li> 
//<ol type="표시할 형태">  --> 표시할 형태 = a, A, i, I
//<ol start="숫자"> --> 시작 값 변경하기
//<ol reversed="reversed"> --> 역순으로 항목 표시


***표 구성 태그들
// <table> : 기본적인 표를 생성해주는 태그
// <tr> : 표의 행을 나타내는 태그
// <th> : 표의 제목 셀을 나타내는 태그
// <td> : 표의 일반 셀을 나타내는 태그
// <caption> : 테이블 제목지정하는 태그
//<colgroup>: 열의 크기를 지정하고 그룹화하는 태그
//<thead>,<tbody>,<tfoot> : 테이블의 틀영역에 맞게 정보를 넣을 수 있는 태그
  ( 각 영역에 지정했기 때문에 순서와 상관없이 정상적으로 테이블을 생성한다.)
//colspan과 rowspan : 해당 열이나 행의 원하는 칸수만큼 병합한다.
 
***영역을 구분하는 태그

// <span> : 줄바꿈이 적용 되지 않음, 옆으로 영역이 붙음(인라인 요소)
