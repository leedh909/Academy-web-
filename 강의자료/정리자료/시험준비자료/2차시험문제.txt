3.	인터페이스와 추상클래스의 차이점에 대해 5가지 기술하시오. (25점)    
1. 인터페이스는 implements로 상속받고, 추상클래스는 extends로 상속받는다.
2. 추상클래스 파일은 재정의된 메소드도 있지만 인터페이스 파일은 정의된 메소드가 없다.

3. 추상클래스는 abstract 메소드가 없을 때도 있지만 인터페이스는 암묵적으로 모든 메소드가 public abstract로 되어있다.

4. 인터페이스는 상속받으면 무조건 정의해줘야한다. 추상클래스는 abstract가 없으면 정의 안해줘도된다.

5. 인터페이스는 다중상속이 가능하지만, 추상클래스는 무조건 한개만 상속 가능하다.

4.	
다량의 데이터를 처리하기 위한 자바 컬렉션 프레임워크가 제공하는 3가지 클래스가 가지는 특징과 대표 클래스명을 2개이상 기술하시오. (25점)

Set :

List :

Map :

   
1. Set : 데이터 중복이 불가능하다. 저장되는 데이터는 순서가 없다.
대표클래스명 : HashSet, TreeSet, LinkedHashSet



2. List : 데이터 중복이 가능하다. 저장되는 데이터는 index번호로 순서화 되어있다.

대표클래스명 : Vector, ArrayList, LinkedList



3. Map : <key,value> 라는 entry 형태로 데이터가 존재한다. 

Key값은 set의 특징처럼 데이터 중복이 불가능하다.

value값은 List의 특징처럼 데이터 중복이 가능하다.

대표클래스명 : HashMap, TreeMap, LinkedHashMap
